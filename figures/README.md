# Figures Directory

This directory contains the generated figures from the brain model comparison analysis.

## Figures

### 1. `brain_prediction_comparison.png`
**Main Performance Comparison Plot**

- **Description**: Layer-wise performance comparison between BERT and CLIP models
- **Content**: 
  - Line plots showing correlation scores for each layer (1-12)
  - BERT (blue) vs CLIP (red) performance
  - Error bars representing 95% confidence intervals
  - Summary statistics box
- **Key Findings**:
  - BERT shows consistently higher correlations across all layers
  - Mean BERT: 0.277, Mean CLIP: 0.201
  - Difference: +0.076 (+37.8% improvement)
  - Best BERT layer: 9 (0.373), Best CLIP layer: 7 (0.284)

### 2. `layer_analysis_detailed.png`
**Detailed Layer-wise Statistical Analysis**

- **Description**: Comprehensive statistical analysis by layer
- **Content**:
  - Layer-wise p-values (statistical significance)
  - Performance difference bars (BERT - CLIP)
  - Statistical power analysis scatter plot
  - Performance distribution histograms
- **Key Findings**:
  - No layer reaches statistical significance (all p > 0.05)
  - P-values range from 0.20 to 0.83
  - Consistent BERT advantage across all layers
  - Largest difference in layer 9 (+0.136)

### 3. `statistical_summary.png`
**Comprehensive Statistical Summary**

- **Description**: Complete statistical analysis overview
- **Content**:
  - Performance comparison with confidence intervals
  - Difference plot with 95% CI
  - Box plots for overall performance distribution
  - Layer-wise significance analysis
  - Effect size visualization (Cohen's d)
  - Summary statistics table
- **Key Findings**:
  - Effect size: Cohen's d = 1.007 (large effect)
  - Paired t-test: t = 1.007, p = 0.351 (not significant)
  - 95% CI for difference: [-0.089, 0.241]
  - Statistical power: 0.15 (inadequate)

## Figure Generation

These figures were generated using the analysis scripts:

```bash
# Generate main comparison plot
python code/analysis/demo_comparison.py

# Generate statistical analysis plots
python code/analysis/statistical_analysis.py

# Generate representational analysis plots
python code/analysis/representational_analysis.py
```
